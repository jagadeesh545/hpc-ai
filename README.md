## ğŸš¢ Deployment to GitHub Pages

### Using gh-pages (Recommended)

This project uses the `gh-pages` npm package for automated GitHub Pages deployment.

**One-time setup:**
1. Ensure `package.json` has `homepage` property pointing to your repository
2. GitHub Pages Settings:
   - Go to Settings â†’ Pages
   - Source: Deploy from a branch
   - Branch: `gh-pages`
   - Folder: `/ (root)`
   - Click Save

**Deploy workflow:**
```bash
# Make changes to source code
git add .
git commit -m "Your changes"
git push origin main

# Deploy (builds and deploys automatically)
npm run deploy
```

**Branch structure:**
- `main` - Your source code (src/, vite.config.js, package.json, etc.)
- `gh-pages` - Auto-generated compiled files (do NOT merge into main)

The `gh-pages` branch is auto-generated by the deploy script. Every deployment replaces its contents. Keep the branches separate!

**Your app is live at:** https://jagadeesh545.github.io/hpc-ai/

### Manual Deployment (Alternative)
1. Build: `npm run build`
2. Push dist/ contents to gh-pages branch manually
3. Configure GitHub Pages to use gh-pages branch

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.

### App not showing styles

- Rebuild: `npm run build`

### GitHub Pages shows 404
- Verify `base: '/hpc-ai/'` in `vite.config.js` matches repo name
